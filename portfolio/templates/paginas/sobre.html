Sobre mim

Meu nome é Fabian Batista da Silva, sou natural de Santo Amaro da Imperatriz e atualmente resido em Palhoça, Santa Catarina. Minha jornada profissional começou a ganhar forma em 2021, quando iniciei minha trajetória na programação com o curso Jovem Programador do Senac-Palhoça. Durante o curso, aprendi lógica de programação e Python, e em seguida, explorei o desenvolvimento web com Django, CSS e HTML. O curso culminou com um projeto final, onde desenvolvi uma API usando Django, complementada com JavaScript, HTML e CSS. Atualmente, trabalho como desenvolvedor na AtendeDireito, onde me especializo em automação de redes sociais e atendimento com inteligência artificial para o setor jurídico. Minha principal responsabilidade é a criação e integração de chatbots e assistentes virtuais que melhoram a comunicação e a eficiência operacional dos clientes. Utilizo tecnologias como Python, JavaScript, n8n e bancos de dados SQL e PostgreSQL para desenvolver soluções robustas e escaláveis. Essas experiências profissionais me permitiram aprimorar minhas habilidades em automação de processos, integração de sistemas e suporte técnico, me motivando a continuar aprendendo e crescendo na área de desenvolvimento web e automação.

Ferramentas

Python

Minha jornada com Python começou com o curso Jovem Programador, oferecido pelo SEPROSC em parceria com o SENAC. Durante um ano de aprendizado intensivo, explorei a linguagem e suas aplicações. Inicialmente, concentrei-me em lógica de programação e Qt Design, estabelecendo uma base sólida. Em seguida, avancei para o desenvolvimento web, aprendendo CSS, HTML e JavaScript, e finalmente, mergulhei no back-end com Django. Atualmente, minha experiência com Python e Django é profundamente enriquecida pela minha atuação como desenvolvedor na automação de processos e atendimento com inteligência artificial. Em meu trabalho atual, sou responsável por criar soluções que integram plataformas de mensagens e otimizam a comunicação com clientes através de chatbots e automações. O Django, em particular, se tornou uma ferramenta essencial no meu dia a dia, permitindo-me desenvolver sistemas robustos e escaláveis. Estou sempre aberto a novas oportunidades de aprendizado e desafios, pois vejo cada um deles como uma chance de continuar evoluindo profissionalmente.

Django

Após dominar os fundamentos do Python, tive a oportunidade de me aprofundar no desenvolvimento web com o framework Django. No curso Jovem Programador, utilizamos Django, juntamente com HTML, CSS e JavaScript, para criar projetos práticos. Em um projeto final desenvolvido em grupo, criamos uma API da tabela TACO. Durante o desenvolvimento, percebemos que o Python por si só não seria suficiente para alcançar um projeto completo, então assumimos o compromisso de aprender JavaScript para adicionar funcionalidades do lado do cliente ao nosso site. Essa experiência me permitiu explorar as capacidades do Django e solidificar meus conhecimentos em desenvolvimento web.
